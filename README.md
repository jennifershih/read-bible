# ReadBible
This project is built with Angular and NestJS, and use open source bible api and openAI API.

## Function
1. select book and chapter, then you can read bible verse.
2. select or input your mood, then AI will generate suitable bible verse for you.

## Built With
* [Angular](Angular.io)
* [Bootstrap](Bootstrap.com)
* [nestjs](nestjs.com)
* [bible API](https://bible-api.com/)
* [openAI API](https://platform.openai.com/overview)
* [vercel](https://vercel.com/)
